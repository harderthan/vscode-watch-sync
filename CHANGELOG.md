# Change Log

All notable changes to the "Watch Sync" extension will be documented in this file.

## [0.0.1] - 2026-01-13

### Added
- Initial release
- Real-time file monitoring using `inotifywait`
- Efficient sync via `rsync`
- Multiple profile support
- Auto-start on VS Code launch
- Status bar integration
- SSH key and password authentication
- Secure credential storage via VS Code SecretStorage
